# FI Quotation Web App v1.5.2

## v1.5.2 Print Layout Fine-tuning

Release นี้ต่อยอดจาก v1.5.1 โดยปรับรายละเอียดหน้า Preview / Print ใบเสนอราคาให้ดูลงตัวขึ้นตาม feedback ล่าสุด

- ตัดข้อความคำอธิบายใต้หัวตารางออกจาก PDF/Print
  - “ส่วนนี้เป็นค่าบริการที่เรียกเก็บเป็นรายเดือน”
  - “ส่วนนี้เป็นค่าบริการที่เรียกเก็บเป็นรายปี”
  - “ส่วนนี้เป็นค่าบริการที่เรียกเก็บครั้งเดียว ณ วันเริ่มใช้งาน”
- ปรับตำแหน่งจำนวนเงินตัวอักษรของแต่ละตารางให้ชิดขอบล่างของกล่องมากขึ้น
- ปรับข้อมูลลูกค้าให้ value ชิด label มากขึ้น
- ปรับข้อมูลบัญชีสำหรับชำระเงินให้ value ชิด label มากขึ้น
- เพิ่มระยะห่างระหว่างตารางค่าบริการรายเดือน/รายปี กับตารางค่าบริการครั้งเดียวให้โปร่งขึ้น

## ไฟล์ใน package

```text

```

## วิธีติดตั้ง

1. Copy ไฟล์ทั้งหมดไปทับใน repo `fi-quotation-web`
2. รอบนี้เป็นการแก้ Frontend/CSS เท่านั้น ถ้าเคยรัน SQL ของ v1.5 และ v1.5.1 แล้ว ไม่ต้องรัน SQL ซ้ำ
3. ถ้ายังไม่เคยรัน v1.5 มาก่อน ให้รันตามลำดับ:

```text
supabase/patch_v1_5.sql
supabase/patch_v1_5_1.sql
```

4. เปิด Live Server แล้วทดสอบหน้า Preview / Print ทั้งแบบรายเดือนและรายปี
5. ถ้าผ่านแล้ว push ขึ้น GitHub Pages

## Push ผ่าน VS Code Terminal

```bash
git status
git add .
git commit -m "Hotfix v1.5.2 print layout spacing"
git push origin main
```

## จุดที่ต้องทดสอบ

- หัวตารางไม่มีคำอธิบายใต้หัวข้อแล้ว
- ใบเสนอราคาแบบรายเดือนยังแสดง 2 ตาราง
- ใบเสนอราคาแบบรายปียังแสดง 2 ตาราง
- จำนวนเงินตัวอักษรของแต่ละตารางอยู่ชิดขอบล่างของกล่องมากขึ้น
- ข้อมูลลูกค้า value ชิด label มากขึ้น
- ข้อมูลบัญชีสำหรับชำระเงิน value ชิด label มากขึ้น
- ระยะห่างระหว่างตารางบนกับตารางล่างโปร่งขึ้น
- Save as PDF แล้วไม่มีหน้า 2 ว่าง
